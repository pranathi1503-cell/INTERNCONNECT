 INTERNCONNECT
 Student Internship Matching Portal
 Overview
 The Student Internship Matching Portal is a web application that helps students find internship opportunities based on their skills, qualifications, and   interests. The system matches student profiles with available internships and provides personalized recommendations.
 Features
Student Registration & Login
Profile Management
Internship Listings
Skill-Based Internship Matching
Search and Filter Internships
Responsive User Interface
Technologies Used
Frontend
HTML
CSS
JavaScript
Backend
Node.js
Express.js
Database
MongoDB
How It Works
Students create an account and complete their profile.
The system stores student details and internship information.
Student skills and interests are matched with internship requirements.
Relevant internship opportunities are displayed to the student.
Objective
To simplify the internship search process by providing students with internship recommendations that match their profiles and career interests.
